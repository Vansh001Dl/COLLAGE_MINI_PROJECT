# Project Documentation

## Overview
This project is a Jupyter Notebook-based application designed for creating image collages. Users can select multiple images and arrange them into a single cohesive collage, making it ideal for personal projects or presentations.

## Features
- Image collage creation
- Customizable layout options
- Automatic resizing of images to fit collage dimensions
- Intuitive Jupyter interface for easy usage
- Supports various image formats (JPEG, PNG, etc.)

## Technologies
- Python
- Jupyter Notebook
- PIL/Pillow
- NumPy

## Installation
To set up the project, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/Vansh001Dl/COLLAGE_MINI_PROJECT.git
   ```
2. Navigate to the project directory:
   ```bash
   cd COLLAGE_MINI_PROJECT
   ```
3. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```
4. Launch Jupyter Notebook:
   ```bash
   jupyter notebook
   ```

## Usage
After launching Jupyter Notebook, open the main notebook file and follow the provided instructions to upload images and customize the collage.
